## Chrome

### 长页面截图
```
1.打开F12
2.按Ctrl + Shift + P
3.输入full size screenshot
4.回车
5.保存截图
```

### 常用插件
Vue.js devtools <br>
WEB前端助手(FeHelper) <br>
React Developer Tools <br>
Adblock Plus <br>
ColorPick Eyedropper <br>
Page Ruler <br>

### 调试动画
```
1.打开F12
2.找Animations选项卡
3.若没有，则点击右上角三点 -> More Tools -> Animations
```
